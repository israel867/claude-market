## Tips

- **Format:** JPG for photos, PNG only if transparency needed
- **Compression:** Run through TinyPNG or Squoosh before uploading — aim for under 200KB per image
- **Consistency:** Shoot before/after pairs at the same angle and lighting
- **Action shots:** Photos of Israel actually working build trust and credibility
- **Vehicles:** Showcase a variety — trucks, SUVs, sedans, luxury cars
- **Cities:** If possible, note which city each photo was taken in for the alt text
